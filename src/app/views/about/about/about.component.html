Service

TLDR

It's divorce predictor trained on public research dataset that runs in-browser for free.

Quick Explainer

The predictor is an artificial neural network (ANN) ; meaning, that it is a brain-inpired piece of software that trains on data to find patterns and make predictions.

In this case, it trained on the UCI Divorce Predictors data set . It was asked to find a pattern between those who got divorced in the dataset and those who didn't. Then this pattern was saved as model on this page. This page then runs this model to make predictions base on the user's responses.
